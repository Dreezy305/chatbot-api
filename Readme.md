### Chatbot API

#### Install packages

```sh
yarn
```

##### Environment variables

[/.env.example](/.env.example)

##### Start database

```sh
yarn start_db
```

##### Start development

```sh
yarn dev
```

##### Start staging/production server

```sh
yarn start
```

##### Start staging

```sh
yarn staging
```

##### Start production

```sh
yarn production
```

##### Test

```sh
yarn test
```
